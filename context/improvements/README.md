# Improvements Directory

This folder groups improvement notes, plans, findings, and implementation specs by topic.

## Groups

- `agent/` — agentic behavior and tool-calling plans.
- `review-backlog/` — grade-it review findings and priority implementation specs.
- `retrieval-performance/` — retrieval failure handling and response-time improvements.
- `indexing-maintenance/` — indexing, stale cleanup, idle load, and maintainability plans/specs.
- `web-ui/` — first-party web UI and stop/cancel behavior plans.
- `security/` — security hardening implementation notes.
- `devops-runtime/` — runtime and development workflow improvements.
